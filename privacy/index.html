개인정보 처리방침

시행일: 2025년 4월 1일

← 효심 홈으로

1. 개인정보의 처리 목적

효심(이하 "앱")은 다음 목적을 위해 최소한의 개인정보를 처리합니다.

  • 가족 생일 등록 및 음력/양력 자동 변환
  • 생일 알림 스케줄링
  • 맞춤 선물 추천 (관계, 나이대, 성별 기반)
  • 홈 화면 및 잠금 화면 위젯 표시

2. 처리하는 개인정보 항목

항목 내용 처리 방식
가족 이름 사용자가 직접 입력한 가족 구성원 이름 기기 내 저장
생년월일 음력 또는 양력 생년월일 기기 내 저장
관계 가족 관계 (예: 아버지, 어머니 등) 기기 내 저장
성별 선물 추천을 위한 성별 정보 기기 내 저장
전화번호 (선택) 원탭 전화/문자 연결용 기기 내 저장
메모 (선택) 가족별 자유 메모 기기 내 저장
핵심 원칙: 효심은 모든 데이터를 사용자의 기기(iPhone)에만 저장합니다. 외부 서버로의 전송, 제3자와의 공유, 클라우드 백업이 일절 없습니다.

3. 개인정보의 처리 및 보유 기간

사용자가 앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다. 별도의 서버 보유 기간이 없습니다.

4. 개인정보의 제3자 제공

효심은 수집한 개인정보를 제3자에게 제공하지 않습니다.

5. 개인정보의 파기

  • 앱 삭제: 앱을 삭제하면 기기 내 모든 데이터가 자동 파기됩니다.
  • 개별 삭제: 앱 내에서 가족 구성원을 삭제하면 해당 정보가 즉시 파기됩니다.

6. 이용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 등록된 가족 정보의 열람, 수정, 삭제
  • 앱 삭제를 통한 전체 데이터 파기
  • 알림 수신 거부 (iOS 설정 > 효심 > 알림)

7. 자동 수집 정보

효심은 앱 사용성 개선을 위해 Firebase Analytics를 통해 익명화된 사용 통계를 수집할 수 있습니다. 이 정보는 개인을 식별할 수 없으며, 다음을 포함합니다:

  • 앱 실행 횟수, 화면 조회 수
  • 기기 유형, OS 버전 (익명)
  • 앱 충돌 로그 (익명)

사용자는 iOS 설정 > 개인정보 보호 및 보안 > 분석 및 개선에서 분석 데이터 공유를 비활성화할 수 있습니다.

8. 개인정보 보호를 위한 기술적 조치

  • 모든 데이터는 기기의 앱 샌드박스 내에 저장되어 다른 앱이 접근할 수 없습니다.
  • SwiftData 프레임워크를 사용하여 안전하게 데이터를 관리합니다.
  • 네트워크 통신 시 HTTPS(TLS) 암호화를 적용합니다.

9. 아동의 개인정보

효심은 만 14세 미만 아동의 개인정보를 수집하지 않으며, 아동이 직접 사용하도록 설계되지 않았습니다.
